Domande frequenti

D. Dove vengono salvati i miei dati registrati?
R. Tutti i dati vengono salvati localmente sul tuo dispositivo. Non vengono mai inviati automaticamente a un server esterno. Quando cambi dispositivo, usa la funzione di backup JSON nelle Impostazioni per esportare i tuoi dati, quindi importali sul nuovo dispositivo.

D. Come annullo il mio abbonamento?
R. iOS: Impostazioni → ID Apple → Abbonamenti → FX Trade Journal.
Android: app Google Play → Menu → Abbonamenti → FX Trade Journal.

D. Il mio acquisto non risulta.
R. Prova il pulsante "Ripristina acquisti" nell'app. Se il problema persiste, contattaci specificando il tuo dispositivo e la versione del sistema operativo.

D. Posso importare lo storico operazioni MT4/MT5?
R. Sì: vai su Impostazioni per importare un file CSV in formato MT4. I formati variano da broker a broker, quindi alcuni file potrebbero non importarsi correttamente.
